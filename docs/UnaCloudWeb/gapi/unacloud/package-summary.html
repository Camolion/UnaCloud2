Package unacloud

In this package are located all controllers, services and domanin classes to manage request from final user
author:
CesarF

  • Class Summary 
    Class Description
    Cluster Entity to represent a Cluster as a group of virtual machine image which have to be deployed together.
    ClusterController This Controller contains actions to manage cluster services: crud and deploy cluster.
    ClusterService This service contains all methods to manage cluster: create and delete cluster.
    ConfigurationController This Controller contains actions to manage configuration services: shows and updates server variables, download and update agent.
    ConfigurationService This service contains all methods to manage configuration: update server variables, manager agent version and download agent files.
    DeployedImage Entity to represent a image that has been deployed in infrastructure, this entity has a relationship with deployment and executions that has been requested to be executed..
    Deployment Entity to represent a Deployment.
    DeploymentController This Controller contains actions to manage deployment services: list, deploy and stop deployments, copy instances and add instances for deployments.
    DeploymentService This service contains all methods to manage deployment: create and delete cluster.
    ErrorController Controller to render error page message
    ExecutionIP Entity to represent an ExecutionIP.
    ExecutionRequest Entity to represent the state of execution in an specified time.
    ExternalCloudAccount Entity to represent an Account in cloud provider.
    ExternalCloudProvider Entity to represent a cloud provider.
    ExternalId Entity to represent an external id in cloud provider.
    HardwareProfile Entity to represent a Hardware Profile base in external cloud providers.
    HardwareProfileService This service contains all methods to manage Hardware profiles: return a list of hardware profiles and query by name filter.
    Hypervisor Entity to represent a Hypervisor.
    HypervisorController This Controller contains actions to manage hypervisor services: deployment crud.
    HypervisorService This service contains all methods to manage Hypervisor: crud hypervisor This class connects with database using hibernate
    IP Entity to represent an IP.
    IPPool Entity to represent a group or pool of IP.
    Laboratory Entity to represent a Laboratory.
    LaboratoryController This Controller contains actions to manage laboratory services: crud for laboratory, physical ip and physical machine.
    LaboratoryService This service contains all methods to manage Laboratory: return a list of hardware profiles and query by name filter.
    MonitorSensor Entity to represent a monitoring sensor.
    MonitorSystem Entity to represent a Monitoring system.
    NetInterface Entity to represent a Network Interface to be configured in a Virtual Machine.
    OperatingSystem Entity to represent an Operating System that is supported by UnaCloud.
    OperatingSystemController This Controller contains actions to manage operating system services: crud for os's.
    OperatingSystemService This service contains all methods to manage Operating System: Operating System crud.
    PhysicalIP Entity to represent an IP which is assign to Physical Machine This entity extends from IP
    PhysicalMachine Entity to represent a Physical Machine.
    PhysicalMachineSensor Entity to represent a sensor for monitoring configured in a virtual machine or physical machine.
    Repository Entity to represent a Repository.
    RepositoryController This Controller contains actions to manage repository services: crud for repositories.
    RepositoryService This service contains all methods to manage Repository: Repository crud methods.
    ServerVariable Entity to represent a Server Variable.
    ServerVariableService This service contains all methods to manage Server Variable: return default allocator and file manager url This class connects with database using hibernate
    User Entity to represent a User in system An User is a UnaCloud user which access to system, upload machines, and configure deployments
    UserController This Controller contains actions to manage User services: crud for user, home, login and logout, profile, user restrictions and change password.
    UserGroup Entity to represent a User Group An User Group is used to create restrictions for a segment of users
    UserGroupController This Controller contains actions to manage User Group services: crud and deploy cluster.
    UserGroupService This service contains all methods to manage User group and users in groups: UserGroup crud methods, and methods to add and remove user from groups.
    UserRestriction Entity to represent a User Restriction An UserRestriction is a restriction for a group or user, like allocator, laboratories and hardware profiles.
    UserRestrictionService This service contains all methods to manage User restriction: these methods returns values like allocator, repository and labs defined to be used for user.
    UserService This service contains all methods to manage User: User crud methods, change password, validate login, and changes api key.
    VirtualMachineExecution Entity to represent a Virtual Machine Execution A Virtual Machine Execution is an execution in a
    VirtualMachineImage Entity to represent a Virtual Machine Image A Virtual Machine Image is a configured file for hypervisor to be deployed in Physical Machines
    VirtualMachineImageController This Controller contains actions to manage virtual machine image services: crud, and services to send messages to agents to copy or deletes images.
    VirtualMachineImageService This service contains all methods to manage User Virtual Machine Image: Crud methods, some methods send tasks in queue This class connects with database using hibernate
