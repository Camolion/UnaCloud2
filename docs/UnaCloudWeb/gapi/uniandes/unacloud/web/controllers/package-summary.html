Package uniandes.unacloud.web.controllers

In this package are located all controllers, services and domanin classes to manage request from final user
author:
CesarF

  • Class Summary 
    Class Description
    AdminController This Controller contains actions to manage Users by Admin User: User crud, set restrictions .
    ClusterController This Controller contains actions to manage cluster services: crud and deploy cluster.
    ConfigurationController This Controller contains actions to manage configuration services: shows and updates server variables, download and update agent.
    DeploymentController This Controller contains actions to manage deployment services: list, deploy and stop deployments, copy instances and add instances for deployments.
    ErrorController Controller to render error page message
    ImageController This Controller contains actions to manage image services: crud and services to send messages to agents to copy or deletes images.
    LaboratoryController This Controller contains actions to manage laboratory services: crud for laboratory, physical ip and physical machine.
    OperatingSystemController This Controller contains actions to manage operating system services: crud for os's.
    PlatformController This Controller contains actions to manage platform services: deployment crud.
    RepositoryController This Controller contains actions to manage repository services: crud for repositories.
    UserController This Controller contains actions to manage User services: home, login and logout, profile and change password.
    UserGroupController This Controller contains actions to manage User Group services: crud and deploy cluster.
