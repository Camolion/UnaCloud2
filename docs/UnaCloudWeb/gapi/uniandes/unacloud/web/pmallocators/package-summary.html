Package uniandes.unacloud.web.pmallocators

In this package are located all allocator algorithms to assign executions to physical machines.
author:
CesarF

  • Class Summary 
    Class Description
    BestFitAllocator Class to execute Best Fit allocator algorithms It sorts physical machines based in available resources, assigns an execution in first machine in list and sorts again.
    BestFitAllocator.PhysicalMachineComparator Class which implements Comparator Compare physical machines base in cores
    ExecutionAllocator Abstract class with main methods to allocate deployments.
    FirstFitAllocator Class to execute First Fit allocator algorithms Assigns all possible executions for each physical machine based in available resources
    FirstFitDecreasingAllocator Class to execute Best Fit allocator algorithms It sorts physical machines based in available resources, assigns an execution in first machine in list and sorts again.
    FirstFitDecreasingAllocator.PhysicalMachineComparator Class which implements Comparator Compare physical machines base in cores
    PhysicalMachineAllocationDescription Class used to represent physical machine resources
    RandomAllocator Class to execute Random allocator algorithms Assigns an execution to a physical machine based in random number.
    RoundRobinAllocator Class to execute Round Robin allocator algorithms Assigns an execution for each physical machine order by physical machine id
    SingletonAllocator Class to execute Singleton allocator algorithms Assigns only one execution for each physical machine
    SortingAllocator Class to execute Sorting allocator algorithm Extends BEST FIT algorithm adding user as a variable in sort process
    SortingAllocator.PhysicalMachineComparator Class which implements Comparator Compare physical machines base in cores and user on machine
